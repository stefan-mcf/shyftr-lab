# ShyftR Current Implementation Status

Status: stable local-first release.

This document is the public truth source for what ShyftR implements today. Ledger files are canonical truth; grid, API, console, reports, and profile artifacts are projections or local append-only writers.

## Current release posture

ShyftR is a stable local-first release for developer-operated cells. It is not a hosted SaaS product, not a multi-tenant production service, and not published as a package release. The supported path is cloning the repository, installing it in a Python 3.11+ environment, and running local cells on synthetic or operator-approved data.
